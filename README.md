## ğŸš€ Getting Started

### Prerequisites

- Node.js (version 18.x or later recommended)
- npm or yarn
- Git
- A text editor (VS Code recommended)

### Initial Setup

1. Clone this repository (if not already done)
2. Install root dependencies:
   ```bash
   npm install
   ```

### Working on Exercises

1. Navigate to the appropriate part folder:

   ```bash
   cd part1
   ```

2. Create a new folder for your exercise:

   ```bash
   mkdir exercise-name
   cd exercise-name
   ```

3. Initialize your project (for React or Node.js):

   ```bash
   # For React apps
   npm create vite@latest . -- --template react

   # For Node.js apps
   npm init -y
   ```

4. Follow the exercise instructions from the course website
